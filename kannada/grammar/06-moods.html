Infinitive

The infinitive form is usually used to indicate like, dislike, wish, or purpose. In English, 'to eat', 'to sleep', 'to sing' are infinitive forms of verbs eat, sleep, sing, respectively.

There are three types of infinitive moods. Each has its own suffix.

  1. Type 1: used to indicate purpose. Suffix is ಅಲಿಕ್ಕೆ (proper) or ಅಕ್ಕೆ (colloquial).
  2. Type 2: used to indicate like/dislike. Suffix is ಅಲು.
  3. Type 3: used before modals. Suffix is ಅ.

Converting verb root to infinitive mood: For ಉ-ending: remove ಉ and add suffix. For non-ಉ-ending: add ಯ್ and then add suffix.

Type 1 examples:

  1. I'm going to the market to buy vegetables. - मैं सब्ज़ी ख़रीदने बाज़ार जा रहा हूँ। - ನಾನು ತರಕಾರಿಗಳು ಕೊಳ್ಳಲಿಕ್ಕೆ marketಗೆ ಹೋಗುತ್ತಾ ಇದ್ದೇನೆ.
  2. One needs a lot of money to study in a University. - University में पढ़ने के लिए बहुत पैसा चाहिए। - Universityಅಲ್ಲಿ ಕಲಿಯಲಿಕ್ಕೆ ತುಂಬಾ ಹಣ ಬೇಕು.

Type 2 examples:

  1. I like to eat sweets. - मुझे मिठाई खाना पसंद है। - ನನಗೆ ಮದುರ ಬಕ್ಷ್ಯ ತಿನ್ನಲು ಇಷ್ಟ.

Type 3 examples:

  1. May I come in? - क्या मैं अंदर आ सकता हूँ? - ನಾನು ಬರ ಬಹುದ?
  2. You cannot go. - आप नहीं जा सकते। - ನೀವು ಹೋಗ ಬಾರದು.

Imperative

Used for commands and requests.

  1. 2nd person singluar: Go there. - वहाँ जाओ. - ಅಲ್ಲಿ ಹೋಗು.
  2. 2nd person plural: Go there. - वहाँ जाइए. - ಅಲ್ಲಿ ಹೋಗಿ.
  3. 1st person singular affirmative: Let me go there. - मैं वहाँ जाता हूँ। / मुझे वहाँ जाने दीजिए। - ನಾನು ಅಲ್ಲಿ ಹೋಗಲಿ.
  4. 1st person singular interrogative: Should I go there? - वहाँ जाऊँ? - ಅಲ್ಲಿ ಹೋಗ್-ಲಾ?
  5. 1st person plural affirmative: Let us go there. - वहाँ जाते हैं। - ಅಲ್ಲಿ ಹೋಗೋಣ.
  6. 1st person plural interrogative: Should we go there? - वहाँ जाएँ? - ಅಲ್ಲಿ ಹೋಗೋಣ್ವಾ?

2nd person imperative form has already been discussed at 04-verbs.html.

Rule to construct first person imperative forms from verb root:

  1. If root ends with ಉ, remove ಉ, otherwise add ಯ್.
  2. Then add suffix: ಅಲಿ for singular affirmative, ಲಾ for singular interrogative, ಓಣ for plural affirmative, ಓಣ್ವಾ for plural interrogative.

Other examples:

  1. Let it be. - रहने दो। - ಇರಲು.
  2. Let it be. - रहने दीजिये। - ಇರಲಿ.

'If' condition: past relative participle + ರ.

  1. If you had gone there, this wouldn't have happened. - तुम वहाँ गए होते तो ये नहीं होता। - ನೀನು ಅಲ್ಲಿ ಹೋಗಿದ್ದರೆ ಅದು ಆಗುತ್ತಿರಲಿಲ್ಲ.
  2. If I had seen him properly yesterday, I could have easily identified him today. - कल उसे मैंने ठीक से देखा होता तो आज मैं उसे आसानी से पहचान सकता था। - ನಿನ್ನೆ ಅವನನ್ನು ನಾನು ಸ್ಪಷ್ಟವಾಗಿ ನೋಡಿದ್ದರೆ, ಇವತ್ತು ಸುಲಭವಾಗಿ ಗುರುತ್ತಿಸ ಬಹುದಾಗಿತ್ತು.
  3. If it rains, we must use the umbrella. - बारिश आए तो छतरी इस्तेमाल करनी चाहिये। - ಮಳೆ ಬಂದರೆ ಛತ್ರಿ ಬಳಸ ಬೇಕು.
  4. If you study hard, you will pass the exam. - तुम मेहनत से पढ़ोगे तो परीक्षा में पास होगे। - ನೀನು ಕಷ್ಟಪಟ್ಟು ಓದಿದರೆ ಪರೀಕ್ಷೆಯಲ್ಲಿ paas ಆಗುತ್ತಿಯ.
  5. If you don't go fast, you will miss the train. - ಬೇಗವಾಗಿ ಹೋಗದಿದ್ದರೆ ನೀನು train ತಪ್ಪಿಸಿಕೊಲ್ಲುತ್ತಿಯ.
  6. What if it rains? - बारिश आई तो क्या होगा? - ಮಳೆ ಬಂದರೆ ಏನಾಗುತ್ತೆ?

'Even if' condition: past relative participle + ರೂ (or ಆಗಿಯೂ instead of ರೂ).
